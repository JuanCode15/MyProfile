Sobre mí

Un poco más de contexto sobre quién soy y cómo trabajo.

Soy Full Stack Developer con experiencia en el desarrollo de aplicaciones web empresariales y soluciones escalables, utilizando Angular, APIs REST, SQL y servicios en la nube de AWS (S3, Lambda, Cognito, API Gateway, entre otros). En Tata Consultancy Services, he participado en proyectos para el sector financiero, colaborando en la construcción de aplicaciones seguras, modernas y de alto rendimiento, contribuyendo tanto en frontend como en backend. En estos proyectos he trabajado en: Desarrollo frontend con Angular, creando interfaces responsivas y dinámicas apoyadas en librerías como PrimeNG y TailwindCSS. Construcción y consumo de APIs REST, optimizando la integración entre sistemas. Gestión y manipulación de datos con SQL, mejorando la eficiencia en consultas y estructuras. Implementación en AWS, utilizando servicios serverless como Lambda, API Gateway y S3 para asegurar escalabilidad y disponibilidad.

Buenas prácticas de desarrollo, aplicando principios SOLID, uso de Git y optimización de dependencias para mantener un código limpio y mantenible. Además, durante mi formación universitaria adquirí conocimientos en diferentes lenguajes y tecnologías como Java (con NetBeans), PHP, MySQL y JavaScript, lo que me ha dado una base sólida para adaptarme a distintos entornos y seguir aprendiendo nuevas herramientas. Mi objetivo profesional es continuar creciendo en el mundo del desarrollo, aportando soluciones innovadoras, escalables y seguras, mientras sigo ampliando mis conocimientos y fortaleciendo mis habilidades técnicas.
